09. Create Raw Transaction Motivation

Why are we having you lookin into raw transactions?

You've seen and used the Bitcoin core wallet which is upon the Bitcoin protocol. It provides us quite a bit of functionality - view transaction, make transactions, generate private keys, etc - but all this is done from the GUI. The GUI, while very user friendly, limits how we can leverage the Bitcoin Protocol.

By understanding how to interact directly with the protocol, we can GUI to help us take advantage of the Bitcoin Protocol to its fullest.

It's like querying Google.com. Using that GUI, we can get quite a bit of information, BUT we can get more functionality if we understand how to directly access Google Protocol layer to build applications on top of it for backend service.

Since we will be building a web app on the blockchain, we would greatly benefit from understanding how to directly leverage the Bitcoin Protocol layer.
