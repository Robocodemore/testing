01. Note to Beta testers

Hey Beta Testers!

Welcome to Lesson 4! You might notice that we have skipped a lesson. Before getting started we wanted to quickly mentioned what's going on.

Based on your feedback we have made some changes to the previous lessons. This resulted in us breaking up Lesson 2 and adding more content across both lessons 2 and 3.

Lesson 2: Bitcoin Core

This lesson is now fully focused on helping you understand the Bitcoin Core and have the best opportunity to begin working with this tool. It includes a few new concepts to help get you up and running.

We have tried to address difficulties you may face and are continuing to add workarounds and alternatives that may help you get started. Thank you for all the support you have provided to help us identify new ways we can help make this more accessible for you.

Lesson 3: Bitcoin Core Debug Console

This lesson contains all of the Bitcoin Core Debug Console topics that were originally in Lesson 2. These are currently the same videos as before, but we have made a ton of revisions based on your feedback and will record new versions shortly.

Lesson 2 New Concepts

While lesson 4 is the bulk of the new content, there have been some additional concepts added to previous lessons. Feel free to go through this content. As always, feedback is appreciated!

  • Mainnet vs Testnet
  • Testnet vs Regnet
  • Bitcoin Core Data warnings
  • Get Testnet Coins
  • Testnet Wallet Transactions
  • Block Explorer

Workspaces

We're excited to announce that we also recently got workspaces up and running.

This is an in classroom interactive environment that allows you to run commands from the classroom. We have not built these features into the classroom yet but wanted to let you know this is coming soon.

This will hopefully help give accessibility to those who are unable to run this software on their local computer.

Thank you!

Thank you so much for your patience with us and the valuable feedback you have provided along the way. It's been amazing having the opportunity to learn from you and we hope you've been able to learn from us.

Good luck with Lesson 4!
