01. Lesson Introduction

Welcome to the lesson on Bitcoin Core!

Bitcoin core is a powerful set of developer tools you can use to help create applications on the bitcoin blockchain.

Lesson Goals

The goal of this lesson is to get you up and running with your own bitcoin core development environment.
Throughout this lesson we’ll introduce bitcoin core, its purpose, and the tools it makes available to help you build and deploy applications.

Once set up, you’ll start to create transactions between wallets using a bitcoin core test environment.
You’ll investigate the transactions made using the online block explorer to get a better idea of how transactions flow through the network.

All this will prepare you to complete the next project, where you’ll start to make transactions using your own private blockchain.

For more details on the upcoming lesson, feel free to read through text below.

When you’re ready we’ll see in you the lesson!

Lesson Overview

You will be able to identify the tooling available to explore the Bitcoin core public blockchain.

Skills

Bitcoin Core, Bitcoin Transactions

Concept Overview

  • Bitcoin Core Overview: Differentiate between bitcoin and bitcoin core, and can explain how they are used by developers.
  • Bitcoin Core Networks: Know when to use each Bitcoin Core Network and navigate between them.
  • Mainnet vs Testnet: Understand the core differences between Bitcoin Core Mainnet and Testnet.
  • Testnet vs Regression Testing: Understand the core differences between Bitcoin Core Testnet and Regnet.
  • Data Warnings: Understand the possible issues with installing Bitcoin Core and ways to address them.
  • Bitcoin Core Getting Started: Set up the Reference Client Bitcoin Core on your desktop.
  • Bitcoin Core Wallet Overview: Find information about their Bitcoin Coin Wallet.
  • Bitcoin Core Demonstration: Demonstrate how to switch networks in the bitcoin core client.
  • Get Testnet Coins: Get your first testnet coins into your personal Bitcoin Core environment.
  • Testnet Wallet Transactions: Setup multiple wallets and make transactions between them.
  • Block Explorer: View your personal transactions using an online block explorer.

When you're ready, feel free to get started with the first concept!
