04. Building a Simple Private Blockchain Motivation

Motivation for Building a Private Blockchain

Now that we have a better understanding of the core concepts behind blockchain data, letâ€™s use the blockchain data model to build our own simple private blockchain.

To understand how blockchain impacts the way we look at data, we need to learn to build our own. We will utilize our classroom workspace with Node.js and NPM to configure our project. If you find working within workspaces cumbersome, feel free to work outside workspaces within your own terminal session with Node.js environment. Node.js version 6+ is preferred.

Here are the steps we will take:

  • Step 1 - Use a testnet transaction to inspire our block data model
  • Step 2 - Build our block class to create new blocks
  • Step 3 - Develop a solution to store blocks utilizing a class constructor
  • Step 4 - Utilizing a third-party library to link block with the SHA256 algorithm
  • Step 5 - Add methods to support block height and timestamps
  • Step 6 - Use a node.js library to wrap levelDB to persist our data
