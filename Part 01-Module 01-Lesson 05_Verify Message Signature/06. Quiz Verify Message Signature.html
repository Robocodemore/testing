06. Quiz: Verify Message Signature

# Step by Step Instructions to solve your Quiz

  • Step 1: Go to your Workspace, and open a New Terminal from the bottom section

  • Step 2: Install the bitcoinjs-message library in order to use the Verify algorithm:

npm install bitcoinjs-message

To confirm if the package has been installed, in the terminal run:

npm list <library name>

  • Step 3: By default, your Workspace will open up a pre-generated file for you called "verifySignature.js".

  • Step 4 : Import/Require the installed library

     const bitcoinMessage = require('bitcoinjs-message');

  • Step 5: In the code, these are already provided:
    1) A pre-defined wallet address
    2) A pre-generated message
    3) Three pre-generated signatures

  • Step 6: Using, the bitcoinMessage.verify(message, address, signature) function, your task is to find out, which signatures Verify to be "TRUE" and which signatures verify to be "FALSE" (hint: use console.log to output the result of the bitcoinMessage.verify function).
